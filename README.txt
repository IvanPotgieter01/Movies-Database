VERSION or DATE: 03/05/2022

AUTHORS: Ivan Potgieter

USER INSTRUCTIONS:
"View Movies" tab:
You are able to view all of the movies in the database. All of the movies are loaded into the combo box. If your movie is selected from the combo box, you are able to delete it from the database.
The Insert button opens a new form from where you are able to insert a new movie into the database.

"Search Movies" tab:
You are able to search the database by entering the name of the movie, scrolling to find the year of the movie, or by selecting the movie's rating from the combo box.
